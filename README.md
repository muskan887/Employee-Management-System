# Employee Management System

## Project Description
Employee Management System is a Java desktop application developed using Java Swing for the graphical user interface, JDBC for database connectivity, and MySQL for data storage. The system allows users to manage employee records efficiently.

## Features
- Add Employee
- View Employee
- Update Employee
- Delete Employee
- Search Employee
- User-friendly GUI

## Technologies Used
- Java
- Java Swing
- JDBC
- MySQL
- Object-Oriented Programming (OOP)

## Database
Database Name: employeemanagementsystem

Tables:
- employee

## How to Run
1. Clone the repository.
2. Import the project into your Java IDE.
3. Create the MySQL database and tables.
4. Update database username and password in `conn.java`.
5. Run `Splash.java` or `Login.java`.

## Project Structure
- Splash.java
- Login.java
- Home.java
- AddEmployee.java
- ViewEmployee.java
- UpdateEmployee.java

- conn.java
